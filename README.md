# Ramadany

Check Iftar times in Ramadan within your terminal.

## Installation

```bash
npx ramadany
```

## Usage

```bash
# Show today's time of iftar and adan
npx ramadany
# Show all time of iftar and adan
npx ramadany --all
# Show iftar time
npx ramadany --iftar
```
